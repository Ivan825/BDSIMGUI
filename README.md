## Setting Up the Environment

1. Create a virtual environment:
   ```bash
   python3 -m venv .venv
``
2. Activate the virtual environment:

    MacOs/Linux:
```bash
    source .venv/bin/activate
```
   Windows:
```bash
    .venv\Scripts\activate
```
3. Install dependencies:
 ```bash
    pip install -r requirements.txt

```
